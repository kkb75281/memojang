# what?

템플릿 엔진으로 HTML 을 PUG 문법으로 작성하면 HTML 로 바꿔주는 기능을 함

# install

terminal 
$ npm install pug

Pug

            bar(1 + 5)
        }
    body
    h1 Pug - node template engine
