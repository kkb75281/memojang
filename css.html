# what?

CSS에 Script의 능력(변수, 함수, 연산, 중첩, 스코프등등)을 덧붙여 확장한 언어

# variable (변수)

css 
@color: #4D926F; 

#header { 
    color: @color; 
} 

h2 { 
    color: @color; 
}

# Mixins (믹스인)

css 
.rounded-corners (@radius: 5px) { 
    border-radius: @radius; 
    -webkit-border-radius: @radius; 
    -moz-border-radius: @radius; 
} 
#header { 
    .rounded-corners; 
} 
#footer { 
    .rounded-corners(10px); 
}

# Nesting (중첩)

css 
#header { 
	h1 { 
        font-size: 26px; 
        font-weight: bold; 
    } 
	p { 
        font-size: 12px; 
        a { 
            text-decoration: none; 
            &:hover { 
                border-width: 1px 
            } 
        } 
    } 
}

# function & operation (함수&연산)

css 
@the-border: 1px; 
@base-color: #111; 
@red: #842210; 
#header { 
    color: @base-color * 3; 
    border-left: @the-border; 
    border-right: @the-border * 2; 
} 
#footer { 
    color: @base-color + #003300; 
    border-color: desaturate(@red, 10%); 
}
